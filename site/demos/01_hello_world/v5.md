# Hello World v5

In this demo, we use Svelte to render data in the Store object and then change
the Cell to see the display update. We're making changes to the Hello World v4
demo.

[base]: # 'Hello World v4'

We swap the React imports for Svelte and the TinyBase Svelte module:

```diff-html
 <script type="importmap">
   {
     "imports": {
       "tinybase": "https://esm.sh/tinybase@",
-      "tinybase/ui-react": "https://esm.sh/tinybase/ui-react@",
-      "tinybase/ui-react-inspector": "https://esm.sh/tinybase/ui-react-inspector@",
-      "react": "https://esm.sh/react@",
-      "react/jsx-runtime": "https://esm.sh/react/jsx-runtime@",
-      "react-dom/client": "https://esm.sh/react-dom/client@"
+      "tinybase/ui-svelte": "https://esm.sh/tinybase@/ui-svelte",
+      "svelte": "https://esm.sh/svelte@"
     }
   }
 </script>
```

The main entry point no longer needs the React imports:

```diff-js
 import {createStore} from 'tinybase';
-import {CellView, Provider} from 'tinybase/ui-react';
-import {Inspector} from 'tinybase/ui-react-inspector';
-import {createRoot} from 'react-dom/client';
-import React from 'react';
+import {mount} from 'svelte';
+import App from './App.svelte';
```

Instead of rendering a React tree, we mount a Svelte component and pass the
Store in as a prop:

```diff-jsx
-createRoot(document.body).render(
-  <Provider store={store}>
-    <CellView tableId="t1" rowId="r1" cellId="c1" />
-    <Inspector />
-  </Provider>,
-);
+mount(App, {target: document.body, props: {store}});
```

The Svelte component uses the Provider and CellView components from the
ui-svelte module:

```svelte file=src/App.svelte
<script>
  import {CellView, Provider} from 'tinybase/ui-svelte';

  let {store} = $props();
</script>

<Provider {store}>
  <CellView tableId="t1" rowId="r1" cellId="c1" />
</Provider>
```

Unlike the React version, there is not yet a Svelte Inspector component, so
this demo just focuses on the live-updating Cell itself.
