# Chart Components (React)

![Chart Components (React)](/shots/styled-chart-react-demo.png)

These demos use the ui-react-dom-charts module to render Store Tables and
Queries ResultTables as simple SVG charts.

The set is organized as a small gallery of chart themes:

| Theme       | Demo                          |

| Basics      | [LineChart][linechart]        |
| Styling     | [Styled LineChart][styled]    |
| Composition | [Composing Charts][composing] |
| Data        | [Sorting And Types][sorting]  |
