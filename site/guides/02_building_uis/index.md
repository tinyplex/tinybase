# Building UIs

These guides cover TinyBase's optional UI modules for building reactive user
interfaces.

Most of this section focuses on the ui-react module, including hooks,
components, React DOM helpers, and context. The Building UIs With Svelte guide
covers the ui-svelte module and its Svelte 5 idioms, the Using Svelte DOM
Components guide covers the ui-svelte-dom module, and the Inspecting Data guide
covers both the React and Svelte inspector modules.

See also the Countries demo, the Todo App demos, and the Drawing demo.
