Hello World (React)

In this demo, we use React to render data in the Store object and then change a Cell to see the display update.

First, since we're running this in a browser, we register some import aliases for esm.sh:

<script type="importmap">
  {
    "imports": {
      "tinybase": "https://esm.sh/tinybase@8.3.0-beta.0",
      "tinybase/ui-react": "https://esm.sh/tinybase@8.3.0-beta.0/ui-react",
      "tinybase/ui-react-inspector": "https://esm.sh/tinybase@8.3.0-beta.0/ui-react-inspector",
      "react": "https://esm.sh/react@^19.2.5",
      "react/jsx-runtime": "https://esm.sh/react@^19.2.5/jsx-runtime",
      "react-dom/client": "https://esm.sh/react-dom@^19.2.5/client"
    }
  }
</script>

In addition to the TinyBase module, we pull in React, ReactDOM, and the TinyBase React modules:

import {createStore} from 'tinybase';
import {CellView, Provider} from 'tinybase/ui-react';
import {Inspector} from 'tinybase/ui-react-inspector';
import {createRoot} from 'react-dom/client';
import React from 'react';

const store = createStore();
const LOGOS = [
  ['https://tinybase.org/favicon.svg', 'TinyBase logo'],
  ['https://tinybase.org/react.svg', 'React logo'],
];

const setTime = () => {
  store.setCell('t1', 'r1', 'c1', new Date().toLocaleTimeString());
};
setTime();

createRoot(document.body).render(
  <Provider store={store}>
    <div id="app">
      <div id="logos">
        {LOGOS.map(([src, alt]) => (
          <img key={alt} src={src} alt={alt} />
        ))}
      </div>
      <div id="value">
        <CellView tableId="t1" rowId="r1" cellId="c1" />
      </div>
    </div>
    <Inspector />
  </Provider>,
);

setInterval(setTime, 1000);

The CellView component takes care of the CellListener for us and re-renders every time the time changes. We also add a compact row of TinyBase and React logos above the value so the demo looks a little more like a React example. Since we're using React, we can also include the Inspector component so you can see how the data is structured. Simply click the TinyBase logo in the corner.
