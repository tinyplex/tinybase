Rolling Dice

These demos demonstrate metrics (such as average dice rolls) and indexes (such as grouping multiple dice rolls).

Averaging Dice Rolls

In this demo, we use a Metrics object to keep a count (and a rolling average) of the values in each Cell in a Store. We roll a dice 48 times and keep track of the average.

Grouping Dice Rolls

In this demo, we use an IndexView component to group each Row of the Store object based on the value in a Cell within it. We roll a dice 48 times and index the rolls by result. We're making changes to the Averaging Dice Rolls demo.
