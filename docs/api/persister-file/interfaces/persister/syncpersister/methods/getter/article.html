Getter methods

This is the collection of getter methods within the SyncPersister interface. There are only two getter methods, getStore and getClient.

getStore

The getStore method returns a reference to the underlying Store that is backing this Persister object.

getClient

The getClient method returns the reference of the Client that the MergeableStore is being persisted via.
