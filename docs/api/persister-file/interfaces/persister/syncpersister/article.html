SyncPersister

The SyncPersister interface is a minor extension to the Persister interface.

It simply provides an extra getClient method for accessing the reference the Client that the MergeableStore is being persisted via.

You should use the createSyncPersister function to create a SyncPersister object.

Since

v5.0.0
