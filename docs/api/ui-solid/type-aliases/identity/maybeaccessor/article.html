MaybeAccessor

The MaybeAccessor type represents a value that can be provided either as a plain value or as a Solid Accessor function.

Thing | Accessor<Thing>

Many ui-solid primitives accept Ids, Store references, and option values as MaybeAccessor values. Passing an Accessor is the idiomatic way to connect a primitive to changing component props or signals, such as () => props.tableId.

When an Accessor is provided, its reactive reads are tracked so the primitive updates when the Accessor value changes.
