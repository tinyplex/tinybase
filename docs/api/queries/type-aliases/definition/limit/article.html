Limit

The Limit type describes a function that lets you specify how many Rows you want in the result Table, and how they should be offset.

Calling this function with one numeric parameter is used to limit the Rows returned, starting from the first.

(limit: number): void
TypeDescription
limitnumber

The number of Rows to return.

returnsvoid

This has no return value.

Calling this function with two numeric parameters is used to offset the start of the results, and then limit the Rows returned.

(
  offset: number,
  limit: number,
): void
TypeDescription
offsetnumber

The number of Rows to skip.

limitnumber

The number of Rows to return.

returnsvoid

This has no return value.

The Limit function is provided as an parameter in the build parameter of the setQueryDefinition method.

A Limit clause can either provide an 'page' offset and size limit, or just the limit. The offset is zero-based, so if you specify 2, say, the results will skip the first two Rows and start with the third.

This is applied after any grouping and sorting.

Examples

This example shows a query that limits a Table to four Rows from its first.

const store = createStore().setTable('pets', {
  cujo: {price: 4},
  fido: {price: 5},
  tom: {price: 3},
  carnaby: {price: 3},
  felix: {price: 4},
  polly: {price: 3},
});

const queries = createQueries(store);
queries.setQueryDefinition('query', 'pets', ({select, limit}) => {
  select('pets', 'price');
  limit(4);
});

queries.forEachResultRow('query', (rowId) => {
  console.log({[rowId]: queries.getResultRow('query', rowId)});
});
// -> {cujo: {price: 4}}
// -> {fido: {price: 5}}
// -> {tom: {price: 3}}
// -> {carnaby: {price: 3}}

This example shows a query that limits a Table to three Rows, offset by two.

const store = createStore().setTable('pets', {
  cujo: {price: 4},
  fido: {price: 5},
  tom: {price: 3},
  carnaby: {price: 3},
  felix: {price: 4},
  polly: {price: 3},
});

const queries = createQueries(store);
queries.setQueryDefinition('query', 'pets', ({select, limit}) => {
  select('pets', 'price');
  limit(2, 3);
});

queries.forEachResultRow('query', (rowId) => {
  console.log({[rowId]: queries.getResultRow('query', rowId)});
});
// -> {tom: {price: 3}}
// -> {carnaby: {price: 3}}
// -> {felix: {price: 4}}
