Creation functions

This is the collection of creation functions within the persister-sqlite-wasm module. There is only one function, createSqliteWasmPersister.

createSqliteWasmPersister

The createSqliteWasmPersister function creates a SqliteWasmPersister object that can persist the Store to a local SQLite database.
