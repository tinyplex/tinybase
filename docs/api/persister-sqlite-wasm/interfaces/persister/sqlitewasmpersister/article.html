SqliteWasmPersister

The SqliteWasmPersister interface is a minor extension to the Persister interface.

It simply provides an extra getDb method for accessing a reference to the database instance the Store is being persisted to.

You should use the createSqliteWasmPersister function to create a SqliteWasmPersister object.

v4.3.14
