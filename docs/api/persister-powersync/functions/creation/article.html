Creation functions

This is the collection of creation functions within the persister-powersync module. There is only one function, createPowerSyncPersister.

createPowerSyncPersister

The createPowerSyncPersister function creates a Persister object that can persist the Store to a local SQLite database that is automatically synced using the PowerSync service.
