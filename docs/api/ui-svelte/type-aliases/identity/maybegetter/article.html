MaybeGetter

The MaybeGetter type represents a value that can be provided either as a plain value or as a reactive getter function.

T | () => T

When a getter function is provided to a hook, the hook's internal $effect will re-run whenever the getter's reactive dependencies change. This is the mechanism that makes Svelte 5 props reactive in hooks.
