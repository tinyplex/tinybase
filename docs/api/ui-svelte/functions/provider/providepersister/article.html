providePersister

The providePersister function registers a Persister with a given Id into the current Provider context.

providePersister(
  persisterId: string,
  persister: AnyPersister,
): void
TypeDescription
persisterIdstring

The Id to register the Persister under.

persisterAnyPersister

The Persister to register.

returnsvoid

This has no return value.
