provideQueries

The provideQueries function registers a Queries object with a given Id into the current Provider context.

provideQueries(
  queriesId: string,
  queries: Queries,
): void
TypeDescription
queriesIdstring

The Id to register the Queries object under.

queriesQueries

The Queries object to register.

returnsvoid

This has no return value.
