onResultRow

The onResultRow function registers a listener that is called whenever a result Row changes.

onResultRow(
  queryId: MaybeGetter<IdOrNull>,
  rowId: MaybeGetter<IdOrNull>,
  listener: ResultRowListener,
  queriesOrQueriesId?: MaybeGetter<undefined | QueriesOrQueriesId>,
): void
TypeDescription
queryIdMaybeGetter<IdOrNull>

The Id of the query to listen to, or null to listen to any query.

rowIdMaybeGetter<IdOrNull>

The Id of the result Row to listen to, or null to listen to any result Row.

listenerResultRowListener

The function to call when the result Row changes.

queriesOrQueriesId?MaybeGetter<undefined | QueriesOrQueriesId>

The Queries object to use, or its Id.

returnsvoid

This has no return value.

Since

v8.1.0
