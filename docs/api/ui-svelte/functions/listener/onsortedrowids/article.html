onSortedRowIds

The onSortedRowIds function registers a listener that is called whenever the sorted Row Ids in a Table change.

onSortedRowIds(
  tableId: MaybeGetter<string>,
  cellId: MaybeGetter<undefined | string>,
  descending: MaybeGetter<boolean>,
  offset: MaybeGetter<number>,
  limit: MaybeGetter<undefined | number>,
  listener: SortedRowIdsListener,
  mutator?: boolean,
  storeOrStoreId?: MaybeGetter<undefined | StoreOrStoreId>,
): void
TypeDescription
tableIdMaybeGetter<string>

The Id of the Table to listen to.

cellIdMaybeGetter<undefined | string>

The Id of the Cell to sort by, or undefined for default order.

descendingMaybeGetter<boolean>

Whether to sort descending.

offsetMaybeGetter<number>

The index of the first Row to include.

limitMaybeGetter<undefined | number>

The maximum number of Rows to include, or undefined for all.

listenerSortedRowIdsListener

The function to call when sorted Row Ids change.

mutator?boolean

An optional boolean indicating the listener mutates Store data.

storeOrStoreId?MaybeGetter<undefined | StoreOrStoreId>

The Store to use, or its Id.

returnsvoid

This has no return value.

Since

v8.1.0
