onPersisterStatus

The onPersisterStatus function registers a listener that is called whenever the status of a Persister changes.

onPersisterStatus(
  listener: StatusListener,
  persisterOrPersisterId?: MaybeGetter<undefined | PersisterOrPersisterId>,
): void
TypeDescription
listenerStatusListener

The function to call when the status changes.

persisterOrPersisterId?MaybeGetter<undefined | PersisterOrPersisterId>

The Persister to use, or its Id.

returnsvoid

This has no return value.
