useResultRow

The useResultRow hook returns a reactive object reflecting a result Row in a result Table, and registers a listener so that any changes will update .current.

useResultRow(
  queryId: MaybeGetter<string>,
  rowId: MaybeGetter<string>,
  queriesOrQueriesId?: MaybeGetter<undefined | QueriesOrQueriesId>,
): {current: Row}
TypeDescription
queryIdMaybeGetter<string>

The Id of the Query (or a getter returning it).

rowIdMaybeGetter<string>

The Id of the Row (or a getter returning it).

queriesOrQueriesId?MaybeGetter<undefined | QueriesOrQueriesId>

The Queries object to use (plain or getter), or its Id.

returns{current: Row}

A reactive object with a current Row property.
