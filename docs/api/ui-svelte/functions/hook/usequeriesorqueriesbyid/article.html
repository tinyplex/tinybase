useQueriesOrQueriesById

The useQueriesOrQueriesById hook is used to get a reference to a Queries object from a Provider context, or have it passed directly.

useQueriesOrQueriesById(queriesOrQueriesId?: MaybeGetter<undefined | QueriesOrQueriesId>): () => Queries | undefined
TypeDescription
queriesOrQueriesId?MaybeGetter<undefined | QueriesOrQueriesId>

The Queries object, its Id, or a getter returning either.

returns() => Queries | undefined

A getter function returning the Queries object, or undefined.
