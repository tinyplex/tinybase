useCheckpointIdsListener

The useCheckpointIdsListener hook registers a listener that is called whenever the Checkpoint Ids change.

useCheckpointIdsListener(
  listener: CheckpointIdsListener,
  checkpointsOrCheckpointsId?: MaybeGetter<undefined | CheckpointsOrCheckpointsId>,
): void
TypeDescription
listenerCheckpointIdsListener

The function to call when Checkpoint Ids change.

checkpointsOrCheckpointsId?MaybeGetter<undefined | CheckpointsOrCheckpointsId>

The Checkpoints object to use, or its Id.

returnsvoid

This has no return value.

Since

v8.1.0
