useSliceRowIdsListener

The useSliceRowIdsListener hook registers a listener that is called whenever the Row Ids in a Slice change.

useSliceRowIdsListener(
  indexId: MaybeGetter<IdOrNull>,
  sliceId: MaybeGetter<IdOrNull>,
  listener: SliceRowIdsListener,
  indexesOrIndexesId?: MaybeGetter<undefined | IndexesOrIndexesId>,
): void
TypeDescription
indexIdMaybeGetter<IdOrNull>

The Id of the Index to listen to, or null to listen to any Index.

sliceIdMaybeGetter<IdOrNull>

The Id of the Slice to listen to, or null to listen to any Slice.

listenerSliceRowIdsListener

The function to call when Slice Row Ids change.

indexesOrIndexesId?MaybeGetter<undefined | IndexesOrIndexesId>

The Indexes object to use, or its Id.

returnsvoid

This has no return value.

Since

v8.1.0
