createQueryIds

The createQueryIds function returns a reactive object reflecting the Ids of the Queries in a Queries object, and registers a listener so that any changes will update .current.

createQueryIds(queriesOrQueriesId?: MaybeGetter<undefined | QueriesOrQueriesId>): {current: Ids}
TypeDescription
queriesOrQueriesId?MaybeGetter<undefined | QueriesOrQueriesId>

The Queries object to use, or its Id.

returns{current: Ids}

A reactive object with a current Ids property.
