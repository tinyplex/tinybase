resolvePersister

The resolvePersister function is used to get a reference to a Persister object from a Provider context, or have it passed directly.

resolvePersister(persisterOrPersisterId?: MaybeGetter<undefined | PersisterOrPersisterId>): () => AnyPersister | undefined
TypeDescription
persisterOrPersisterId?MaybeGetter<undefined | PersisterOrPersisterId>

The Persister object, its Id, or a getter returning either.

returns() => AnyPersister | undefined

A getter function returning the Persister object, or undefined.
