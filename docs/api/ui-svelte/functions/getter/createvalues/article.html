createValues

The createValues function returns a reactive object reflecting the Values in the Store, and registers a listener so that any changes will update .current.

createValues(storeOrStoreId?: MaybeGetter<undefined | StoreOrStoreId>): {current: Values}
TypeDescription
storeOrStoreId?MaybeGetter<undefined | StoreOrStoreId>

The Store to use, or its Id in a Provider context.

returns{current: Values}

A reactive object with a current Values property.
