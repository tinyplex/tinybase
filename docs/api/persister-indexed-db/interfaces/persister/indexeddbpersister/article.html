IndexedDbPersister

The IndexedDbPersister interface is a minor extension to the Persister interface.

It simply provides an extra getDbName method for accessing the unique key of the IndexedDB the Store is being persisted to.

Since

v4.3.14

getDbName

The getDbName method returns the unique key of the IndexedDB the Store is being persisted to.
