HasIndexListener

The HasIndexListener type describes a function that is used to listen to Index definitions being added or removed.

(
  indexes: Indexes,
  indexId: Id,
  hasIndex: boolean,
): void
TypeDescription
indexesIndexes

A reference to the Indexes object that changed.

indexIdId

The Id of the Index that changed.

hasIndexboolean

Whether the Index now exists.

returnsvoid

This has no return value.

A HasIndexListener is provided when using the addHasIndexListener method. See that method for specific examples.

When called, a HasIndexListener is given a reference to the Indexes object, the Id of the Index that changed, and whether it now exists.

v9.1.0
