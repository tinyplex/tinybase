Listener methods

addClientIdsListener

The addClientIdsListener method registers a listener function with the WsServer that will be called whenever there is a change in the clients connected to a path that a WsServer is handling. Read more.

addPathIdsListener

The addPathIdsListener method registers a listener function with the WsServer that will be called whenever there is a change in the active paths that a WsServer is handling. Read more.

delListener

The delListener method removes a listener that was previously added to the WsServer. Read more.
