PathIdsListener

The PathIdsListener type describes a function that is used to listen to changes of active paths that a WsServer is handling.

(
  wsServer: WsServer,
  getIdChanges: GetIdChanges,
): void
TypeDescription
wsServerWsServer

A reference to the WsServer.

getIdChangesGetIdChanges

A function that returns information about the path Id change.

returnsvoid

This has no return value.

A WsServer listens to any path, allowing an app to have the concept of distinct 'rooms' that only certain clients are participating in. As soon as a single client connects to a new path, this listener will be called with a GetIdChanges callback that you can use to determine its Id.

When the final client disconnects from a path, it will be called again with a GetIdChanges callback that you can again use to determine its Id.

A PathIdsListener is provided when using the addPathIdsListener method. See that method for specific examples.
