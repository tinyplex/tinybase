addWillDelTablesCallback

The addWillDelTablesCallback method registers a WillDelTablesCallback that will be called before all Tables are deleted from the Store.

addWillDelTablesCallback(callback: WillDelTablesCallback): Middleware
TypeDescription
callbackWillDelTablesCallback

The WillDelTablesCallback to register.

returnsMiddleware

A reference to the Middleware object, for chaining.

The callback returns true to allow the deletion or false to prevent it. Multiple callbacks can be registered and they are called sequentially. If any callback returns false, the deletion is prevented.

Example

This example registers a callback that prevents deleting all Tables from the pet store.

import {createMiddleware, createStore} from 'tinybase';

const store = createStore();
const middleware = createMiddleware(store);

store.setTables({pets: {fido: {species: 'dog'}, felix: {species: 'cat'}}});

middleware.addWillDelTablesCallback(() => false);

store.delTables();
console.log(store.getTables());
// -> {pets: {fido: {species: 'dog'}, felix: {species: 'cat'}}}

middleware.destroy();
