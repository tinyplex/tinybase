Configuration type aliases

This is the collection of configuration type aliases within the persister-partykit-client module. There is only one type alias, PartyKitPersisterConfig.

PartyKitPersisterConfig

The PartyKitPersisterConfig type describes the configuration of a PartyKit Persister on the client side.
