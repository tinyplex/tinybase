XAxis

The XAxis component configures the x axis of a CartesianChart component.

XAxis(props: XAxisProps): ComponentReturnType
TypeDescription
propsXAxisProps
returnsComponentReturnType

It is a configuration child rather than a separately rendered SVG element: include zero or one XAxis component in a CartesianChart component's children. If omitted, the x axis is inferred from the chart's series.
