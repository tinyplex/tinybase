ChartProps

The ChartProps type describes the props that are used to configure the chart element that renders a chart.

{className?: string}
TypeDescription
className?string

An optional string that will be used as the class attribute of the chart's SVG element. This can be used as the root selector for styling chart internals, as shown in the Using Charts guide.

v8.5.0
