XAxisProps

The XAxisProps type describes the props that configure the x axis of a CartesianChart component.

{
  className?: string;
  title?: string;
  min?: number;
  max?: number;
  ticks?: readonly number[];
  tickCount?: number;
  tickFormatter?: (tick: boolean | number | string) => string;
}
TypeDescription
className?string

An optional string that will be added to the class attribute of the x-axis SVG group element.

title?string

An optional title to use for the x axis, defaulting to the x Cell Id, or to a combined title when multiple series use different x Cell Ids.

min?number

An optional minimum x-axis bound for continuous numeric x values.

max?number

An optional maximum x-axis bound for continuous numeric x values.

ticks?readonly number[]

Optional numeric tick values to use for a continuous x axis.

tickCount?number

An optional preferred number of ticks to render on a continuous x axis.

tickFormatter?(tick: boolean | number | string) => string

An optional function for formatting x-axis tick labels.

The x axis is inferred by default. Use an XAxis component child when you want to override its title, numeric bounds, tick values, tick count, tick labels, or SVG class name.
