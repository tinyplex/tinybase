SliceInHtmlTable

The SliceInHtmlTable component renders the contents of a Slice as an HTML

<table> element, and registers a listener so that any changes to that result will cause a re-render. [Read more](/api/ui-react-dom/functions/indexes-components/sliceinhtmltable/).
