AutomergePersister

The AutomergePersister interface is a minor extension to the Persister interface.

It simply provides an extra getDocHandle method for accessing the Automerge document handler the Store is being persisted to.

Since

v4.3.14

getDocHandle

The getDocHandle method returns the Automerge document handler the Store is being persisted to.
