Content

The Content type describes both the Tables and Values in a Store.

[Tables, Values]

It is an array of two objects, representing tabular and keyed value content.

Example

The following is a valid Content array:

[
  {
    "pets": {
      "fido": {
        "sold": false,
        "price": 4,
      },
      "felix": {
        "sold": true,
        "price": 5,
      },
    },
  },
  {
    open: true,
    employees: 3,
  },
]
