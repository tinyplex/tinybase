Schema type aliases

TablesSchema

The TablesSchema type describes the tabular structure of a Store in terms of valid Table Ids and the types of Cell that can exist within them. Read more.

NoTablesSchema

The NoTablesSchema type is a TablesSchema-like type for when one has not been provided. Read more.

OptionalTablesSchema

The OptionalTablesSchema type is used by generic types that can optionally take a TablesSchema. Read more.

CellSchema

The CellSchema type describes what values are allowed for each Cell in a Table. Read more.

ValuesSchema

The ValuesSchema type describes the keyed Values that can be set in a Store and their types. Read more.

NoValuesSchema

The NoValuesSchema type is a ValuesSchema-like type for when one has not been provided. Read more.

OptionalValuesSchema

The OptionalValuesSchema type is used by generic types that can optionally take a ValuesSchema. Read more.

ValueSchema

The ValueSchema type describes what values are allowed for keyed Values in a Store. Read more.

NoSchemas

The NoSchemas type is used as a default by generic types that can optionally take either or both of a TablesSchema and ValuesSchema. Read more.

OptionalSchemas

The OptionalSchemas type is used by generic types that can optionally take either or both of a TablesSchema and ValuesSchema. Read more.
